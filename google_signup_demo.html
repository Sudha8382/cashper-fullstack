üîå API Endpoint

POST /api/auth/google-login

Request Body:

{ "token": "google-oauth-token-from-frontend" }

Response (Success):

{ "access_token": "eyJhbGciOiJIUzI1NiIs...", "token_type": "bearer", "user": { "id": "673abc...", "fullName": "John Doe", "email": "john@gmail.com", "phone": "", "isEmailVerified": true, "isPhoneVerified": false } }

What Backend Does:

  • 1. Verifies Google token with Google servers
  • 2. Extracts user info (name, email, Google ID)
  • 3. Checks if user exists in database
  • 4. Creates new account OR logs in existing user
  • 5. Generates secure JWT token
  • 6. Returns token and user data
